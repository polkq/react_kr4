# Technology Tracker - Трекер изучения технологий

Проект создан в рамках практического занятия №19 по курсу "Фронтенд и бэкенд разработка".

## Описание

Приложение для отслеживания прогресса изучения различных технологий. Позволяет визуализировать статус изучения каждой технологии и общий прогресс.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

3. Откройте браузер по адресу, указанному в консоли (обычно http://localhost:5173)

## Структура проекта

```
src/

  ├── App.css
```

## Функциональность

- Отображение списка технологий с различными статусами:
  - Изучено (completed)
  - В процессе (in-progress)
  - Не начато (not-started)
- Статистика по всем технологиям
- Визуальный прогресс-бар с процентом выполнения
- Адаптивный дизайн
